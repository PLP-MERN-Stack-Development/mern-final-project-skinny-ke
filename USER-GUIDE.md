### Managing Team Members

1. Go to your workspace settings

3. Click **"Invite Member"**
4. Enter email addresses and assign roles:

## üìã Task Management

### Creating Tasks

#### Quick Task Creation
1. Click the **"+"** button in any workspace
2. Enter task title
3. Optionally add description, assignee, and due date
4. Task is created instantly and visible to your team

#### Detailed Task Creation
1. Click **"New Task"** for full task creation form
2. Fill in comprehensive details:
   - **Title**: Clear, descriptive task name
   - **Description**: Detailed task requirements
   - **Priority**: Low, Medium, High, Urgent
   - **Assignees**: Select team members
   - **Due Date**: Set deadline with time
   - **Estimated Hours**: Time estimation
   - **Tags**: Categorize with labels
   - **Dependencies**: Link to other tasks
   - **Subtasks**: Break down into smaller steps

### Task Views

#### Kanban Board View
- **Columns**: To Do, In Progress, Review, Done
- **Drag & Drop**: Move tasks between columns
- **Quick Actions**: Right-click for context menu
- **Filters**: Filter by assignee, priority, due date, tags

#### List View
- **Compact Display**: All tasks in a scrollable list
- **Sorting Options**: Sort by due date, priority, assignee
- **Bulk Actions**: Select multiple tasks for batch operations
- **Quick Edit**: Inline editing for fast updates

#### Calendar View
- **Timeline Visualization**: See tasks on a calendar
- **Month/Week/Day Views**: Different time perspectives
- **Drag to Reschedule**: Move tasks by dragging
- **Color Coding**: Priority and status indicators

### Task Details and Collaboration

#### Task Detail Panel
- **Full Description**: Rich text formatting supported
- **Activity Log**: Complete history of changes
- **Time Tracking**: Log actual hours worked
- **File Attachments**: Upload and manage files

#### Comments and Mentions
- **Rich Text Comments**: Format text, add links, code blocks
- **@Mentions**: Tag team members for notifications
- **File Attachments**: Attach files to comments
- **Reactions**: Add emoji reactions to comments
- **Threaded Replies**: Reply to specific comments

#### Subtasks
- **Break Down Work**: Divide tasks into manageable pieces
- **Progress Tracking**: Mark subtasks as complete
- **Assignment**: Assign subtasks to different team members
- **Dependencies**: Set subtask relationships

### Task Filters and Search

#### Advanced Filtering
- **Status**: Filter by task status
- **Priority**: Filter by priority level
- **Assignee**: Filter by team member
- **Due Date**: Filter by time periods (Today, This Week, Overdue)
- **Tags**: Filter by labels
- **Creator**: Filter by who created the task

#### Search Functionality
- **Full-Text Search**: Search across titles, descriptions, comments
- **Advanced Operators**: Use AND, OR, NOT for complex queries
- **Saved Searches**: Save frequently used search filters
- **Recent Searches**: Quick access to recent searches
